# Scripts

Utility scripts for homelab management.

## Contents

| Directory | Description |
|-----------|-------------|
| `update_portainer_script/` | Update Portainer CE to latest LTS |
| `install_promtail_node-exporter_script/` | Install Promtail and Node Exporter on Debian hosts |

## Usage

Each script is self-contained. Copy to target host and run with appropriate permissions.

```bash
chmod +x script.sh
./script.sh
```

See individual README files for specific requirements and options.
